##  Live Deployment

**Deployed Link** : [link](https://interview-management-system-qkit.vercel.app/)
The application is deployed across a modern 3-tier architecture:
- **Frontend (Client):** Vercel
- **Backend (API):** Render
- **Database:** MongoDB Atlas

***

##  Technology Stack

### Frontend
* **Framework**: React 18, [Next.js](https://nextjs.org/) (App Router)
* **Styling**: Tailwind CSS
* **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (Radix UI)
* **API Client**: Axios
* **Icons & Typology**: Lucide React, Google Fonts (Nunito, Inter)

### Backend
* **Runtime**: [Node.js](https://nodejs.org/)
* **Framework**: [Express.js](https://expressjs.com/)
* **Authentication**: JSON Web Tokens (JWT) & bcryptjs
* **Database**: MongoDB via Mongoose ORM
* **Architecture**: MVC (Models, Views/Routes, Controllers)
